# 🌤 Weather App (Login + Dashboard)

A simple Weather Web Application built using HTML, CSS, and JavaScript.  
It includes login authentication and a live weather dashboard using WeatherAPI.

---

## 🚀 Features

- 🔐 Login system using local JSON file
- 🌍 Search weather by city name
- 🌡 Live temperature data
- 📅 3-day weather forecast
- 🎨 Modern glassmorphism UI
- 🚪 Logout functionality
- 🔒 Protected dashboard access

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- WeatherAPI
- Git & GitHub

---

## 🔥 Project Flow

1. Login page opens
2. User enters credentials
3. Data checked from `users.json`
4. Redirect to dashboard
5. Search city weather
6. View live weather + forecast
7. Logout anytime
